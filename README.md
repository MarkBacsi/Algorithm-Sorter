# Algorithm Sorter

A visual C# application that demos various different sorting algorithms in action.

## What It Does

This project visualizes how different sorting algorithms work, making it easier to understand and compare their performance. Watch as the algorithms sort data in real time.

## Getting Started

### Requirements

- .NET Framework or .NET Core.
- Visual Studio Code.

### Running the Project

1. Clone the repository onto main pc.
2. Open the whole code in Visual Studio.
3. Build and run the solution.

## Features

- Visual representation of various different sorting algorithms that can be selected from.
- Compare different algorithm speeds and how they go.
- Step-by-step visualization.

## Algorithms Included

- Bubble Sort
- Quick Sort
- Merge Sort

## How to Use

1. Launch the application.
2. Select an algorithm from the menu.
3. Click "Confirm" and watch the algorithm go.
4. Adjust speed settings to see the process in more detail.
